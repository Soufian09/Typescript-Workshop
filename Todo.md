## Let's go 

Supprimez le fichier ```App.css``` , ensuite sur votre fichier ```App.tsx``` , coller ce bout de code ci-dessous .

``` import React, { Component } from "react";

class App extends Component {
  render() {
    return (
      <div>
        <h2>Hello React TS!</h2>
      </div>
    );
  }
}

export default App;```
